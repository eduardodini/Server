## Node Install:
 ```
 brew install node
 ```

 ## NPM install
 ```
 npm install -g npm
 ```
 OR
 ```
 brew install npm
 ```

 ## NPM initializer
 ```
 npm init -y
 ```

 ## Express Install
 ```
 npm i express@4.17.3 --save -E
 ```

 ### Typescript version
 ```
 npm i @types/express@4.17.13 -D -E
 ```
